Retro Starfighter Trench Run

Una escena retro en canvas para simular el vuelo de una nave por una trinchera wireframe a toda velocidad.

Estado

Preparado para entrar en la trinchera

Controles

  • ← / A: mover a la izquierda
  • → / D: mover a la derecha
  • ↑ / W: subir
  • ↓ / S: bajar
  • R: reiniciar la escena

Mecánica

  • La nave se mueve dentro de una zona limitada.
  • La trinchera avanza hacia el jugador.
  • Los obstáculos aparecen al fondo y se acercan.
